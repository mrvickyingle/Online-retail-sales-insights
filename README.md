# Online-retail-sales-insights
Exel based retail sales analysis project with cleaned data, pivot tables, dashboard, insights and business recommandations
Excel Data Cleaning & Refinement Project
📌 Project Overview

This project focuses on cleaning and refining a raw Excel dataset to make it accurate, consistent, and ready for analysis. The dataset was processed using Microsoft Excel by applying various data cleaning techniques and validation checks.

🎯 Objective

The main objective of this project is to:

Remove inconsistencies from the raw data.
Improve data quality and accuracy.
Prepare the dataset for reporting, visualization, and data analysis.
🛠️ Tools Used
Microsoft Excel
🧹 Data Cleaning Process

The following cleaning and transformation steps were performed:

Removed duplicate records.
Handled missing or blank values.
Corrected inconsistent text formatting.
Standardized date formats.
Verified numeric data types.
Removed unnecessary spaces using TRIM().
Corrected text capitalization using PROPER(), UPPER(), and LOWER() where required.
Created new calculated columns.
Validated data for consistency.
Organized and structured the dataset for analysis.
📊 New Columns Added

The cleaned dataset includes the following derived columns:

Is Cancelled
Transaction Type
Quantity Status
Price Status
Total Sales
Year Month
✅ Data Quality Improvements

After cleaning, the dataset is:

Free from duplicate records.
Consistent in formatting.
Ready for Pivot Tables and Charts.
Suitable for Excel dashboards.
Ready for SQL, Power BI, or Python analysis.
